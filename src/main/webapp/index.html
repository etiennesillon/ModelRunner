ModelRunner is an open source, no-code, speech-enabled, data management platform.

What does it mean? Let's break it down:

  • open source: all the code is available on GitHub with a MIT License.
  • no-code: just use the graphical editor to design a model of your data and click on Deploy! All the code needed is already written and tested.
  • speech enabled: if you don't like using a keyboard, just use your voice to design and deploy your model and start managing your data, also just using your voice!
  • data management platform: use the out-of-the-box web application and the speech user interface to create and query your data.
